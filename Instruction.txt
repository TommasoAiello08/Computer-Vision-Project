2) The eight point algorithm for essential matrix estimation (4 pts)
Implement the eight-point algorithm to estimate the essential matrix and test it on the pair of images you
have captured.
You may:
• Use SIFT with robust fitting (e.g., RANSAC) to extract correspondences, or
• Manually annotate correspondences.
Requirements:
• Use your own implementation of the eight-point algorithm.
• Adapt the algorithm to calibrated images by using the camera calibration matrix estimated in the
previous step.
• Visualize the results by plotting all epipolar lines in the first image.
• Use a geometric error (e.g., epipolar constraint error or Sampson distance) to evaluate how well the
estimated matrix models the epipolar geometry.
• Compare your results with the five-point algorithm provided in OpenCV.
Provide enough visualizations and carefully describe all the steps. Comments whether there are difference
between your results and the one attained with the five-point algorithm. Why?
